# Ludoteca Viva – pastel UI

Interfaz Streamlit para la Google Sheet viva de la ludoteca.

Incluye:
- Estética pastel.
- Sidebar de filtros.
- Métricas superiores.
- Tabla principal con miniaturas, BGG ID y link a BoardGameGeek.
- Paneles inferiores.
- Gráficos simples.
- Sin segunda fila de botones de acción.

Archivos:
- app.py
- requirements.txt
- README.md
- bgg_catalog.json
